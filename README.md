## Что такое алгоритм

Алгоритм - это точная конечная последовательность действий для решения задачи

Свойства алгоритма:
- Дискретность - состоит из отдельных шагов
- Детерминированность - каждый шаг однозначно определён
- Конечность - завершается за конечное число шагов
- Массовость - применим для целого класса задач
- Результативность - всегда приводит к результату

## Блок-схемы

Блок-схема - это графическое представление алгоритма

Основные элементы:
- Овал - начало или конец алгоритма
- Прямоугольник - выполнение действия
- Ромб - условие (да/нет)
- Параллелограмм - ввод или вывод данных
- Стрелка - направление выполнения
